# Claude Code Sandbox

This repository is an experimental space for exploring AI-driven software development and AI team workflows using Claude Code.

## Purpose

Each subdirectory contains an independent experiment. Experiments may involve:
- Autonomous AI agents building software
- Multi-agent coordination and collaboration
- AI-assisted development workflows
- Claude Code SDK and API integrations

## Repository Structure

```
/
├── CLAUDE.md          # This file
├── README.md
└── <experiment-name>/ # Each experiment lives in its own subdirectory
```

## Working with Experiments

Each experiment subdirectory may have its own `CLAUDE.md` with context specific to that experiment. Start there when working inside a subdirectory.

## General Guidelines

- Keep experiments isolated in their own subdirectories
- Document the goal and approach in each experiment's README or CLAUDE.md
- Prefer small, focused experiments over large monolithic ones
